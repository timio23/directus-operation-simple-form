# Simple Form Operation

Submit JSON data as a Form to any URL along with custom headers.

## Example Input

```
{
    "first_name": "Value",
    "last_name": "Value",
    "number_field": 123,
    "radio_field": "Value",
    "checkbox": "Value",
    "mutiple_choice": [
        "Option 1",
        "Option 2"
    ]
}
```
